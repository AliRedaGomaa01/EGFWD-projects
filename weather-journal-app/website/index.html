Weather Journal App

If you want to get weather of some city , please , follow these steps
 1) search on google for
   1-1) city zip code Ex. western California zip code
  1-2) country iso alpha-2 code , Ex. America iso alpha-2 code
 2) enter the data in the proper input box using this format cityZipCode,countryCode Ex. 94040,us
 3) It's optional to input your feeling
 4) press the button "Generate"

It may take at most 5 seconds
